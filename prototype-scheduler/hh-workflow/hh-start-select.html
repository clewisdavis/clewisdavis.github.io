Choose Start Date

Select the date when new or rescheduled lessons should start appearing on the Planner. Any incomplete or overdue lessons will also be rescheduled starting on this date. All completed lessons will remain on the date they were originally marked complete.

Below you can select today (recommended) or a future date (advanced) as the start date. Do not set a date in the past. If you choose to set the date for each course individually, enter the start date in the New Start Date column.

Choosing a Start Date

  • If a course's start date is today and your student has already completed lessons, note that new lessons may also be added to today's date.
  • Setting individual start dates per course?
    • Custom start dates may not be in the past.
    • Custom start dates may not be prior to the original course start date.
